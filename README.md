# C言語開発環境（Docker）

## パッケージ一覧

- Ubuntu：22.10
- build-essential
- cmake
- clang
- nano
- gdb

## 構築方法

### for Windows

#### 前提条件

- WSL2の有効化
- Docker Desktop for Windowsのインストール
　Docker Desktop Installer.exeをダウンロードし、起動
  
  WSL2 installation is in completeが表示されたら、リンクを踏み、x64マシン用WSL2　Linuxカーネル更新プログラムパッケージをダウンロードし、起動
